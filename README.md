Informe del avance del proyecto

Este en un programa que maneja el inventario de una tienda de telofonos moviles, l inventario esta separado en el inventario
de articulos y de telofonos moviles y tablets.

Los inventarios estan bien, estan conectados a una base de datos local de MySQL, por lo que para su funcionamiento 
debes crear una base de datos local.

Dejare el archivo con la tablas correspondientes y los campos por si deseas modificar la información, cualquier mejora o
sugerencia por favor escribirme al correo valenciawaner@gmail.com.

Soy estudiante de ingeniería de sistemas, principiante por lo que sería de gran ayuda vuestra colaboración en este proyecto.

La parte de busqueda esta a medias debido a que se presenta un error al hacer la consulta y no se debido a que si hago los
debidos procedimientos.
